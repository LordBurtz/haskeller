Scala 3 REST Service

Welcome to the Todo REST API built with Scala 3, Http4s, and Cats Effect.

API Endpoints

List all todos

GET /todos

Get a specific todo

GET /todos/{id}

Create a new todo

POST /todos

Example request body:

{
  "title": "Learn Scala",
  "completed": false
}

Update a todo

PUT /todos/{id}

Example request body:

{
  "title": "Learn Scala 3",
  "completed": true
}

Delete a todo

DELETE /todos/{id}

Powered by Scala 3, Http4s, and Cats Effect
